# Installation

install all the required libs 


```pip install requests```

```pip install colorama```

```pip install pathlib```






Run:
```--:> python USV.py```

#
<h4>The Git way :</h4>

```--:> git clone https://github.com/suegdu/USV.git```

```--:> cd USV```

```--:> pip install -r requirements.txt```

Run:
```--:>python USV.py```
